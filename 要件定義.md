# KOUEI Trading Website 要件定義書

## プロジェクト概要

### 目的
既存のkjt.co.jpウェブサイトを完全に再構築し、モダンで使いやすく、管理しやすいコーポレートサイトを構築する。

### 基本方針
- **ゼロから完全再構築**: 既存サイトの完全なリデザイン・リビルド
- **モダンUI/UX**: 今風でかっこいいデザインとインタラクション
- **YAML CMS**: 簡単に更新できるコンテンツ管理システム
- **レスポンシブ対応**: モバイル・タブレット・デスクトップ完全対応
- **グローバル対応**: 英語サポート機能の実装

## 技術要件

### フロントエンド技術スタック
- **フレームワーク**: Next.js 14 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS + カスタムCSS
- **アイコン**: Heroicons (SVG)
- **フォント**: システムフォント + Google Fonts

### バックエンド・CMS
- **CMS**: YAML ベースのファイルシステムCMS
- **データローダー**: サーバーサイド YAML パーサー (js-yaml)
- **コンテンツ管理**: 各種データを YAML ファイルで管理
- **メール機能**: Nodemailer (お問い合わせフォーム)

### デプロイ・開発環境
- **ホスティング**: Vercel
- **CI/CD**: GitHub Actions
- **バージョン管理**: Git/GitHub
- **開発ツール**: Claude Code (AI コーディング支援)

## 機能要件

### 1. トップページ
- **メインビジュアル**: グローバルマップ風デザイン
- **統計データ表示**: 取引企業数、展開国数、実績年数
- **企業ビジョン・ミッション**: Vision/Mission の表示
- **海外販売先**: 6地域・100+国の取引実績表示
- **トピックス**: 最新6件のニュース・更新情報
- **CTA**: お問い合わせへの誘導

### 2. 販売事業ページ (/sales)
- **カード型ナビゲーション**: 7つの事業部門を魅力的なカードで表示
  - 日本国内販売 (青)
  - ベトナム事業 (緑)
  - ミャンマー事業 (紫)
  - 海外プロジェクト向け輸出 (オレンジ)
  - 海外工場向け輸出 (ティール)
  - 輸出入実績 (インディゴ)
  - 海外取引支援 (赤)
- **ホバーアニメーション**: カード浮き上がり効果
- **事業概要**: 主力商品・サービス内容の説明
- **地域別販売実績**: YAML データからの動的表示

### 3. 取扱メーカーページ (/manufacturers)
- **YAML CMS**: 900+社のメーカーデータをYAML管理
- **カテゴリ分類**: 20のカテゴリによる整理
  - 作業工具、荷役運搬機器、測定機器、油空圧・液体機器、電気、安全具、切削工具、溶接関連、仮設機器、塗装、油圧・空圧機器、化学品、環境機器、ボルト・ナット、橋梁・土木、軸受、ゴム製品、金属加工品、海外メーカー
- **取引ステータス**: 「取引」バッジでパートナーシップ表示
- **検索・フィルタ**: カテゴリ別表示

### 4. 会社概要ページ (/about)
- **4組織選択カード**: 
  - 恒栄トレーディング (日本本社・東京) - 青
  - 恒栄ベトナム (ベトナム拠点・ホーチミン) - 緑
  - 恒栄ミャンマー (ミャンマー支社・ヤンゴン) - 紫
  - 恒栄バングラデシュ (バングラデシュ事務所・ダッカ) - オレンジ
- **グループ概要**: ビジョン、ミッション、実績
- **沿革**: タイムライン形式での会社史
- **詳細ページ**: 各組織の詳細情報ページ

### 5. お問い合わせページ (/contact)
- **問い合わせフォーム**: 名前、メール、件名、内容
- **メール送信機能**: Nodemailer による自動送信
- **英語サポート**: English Support Available
- **連絡先情報**: 各拠点の連絡先表示

## コンテンツ管理システム (YAML CMS)

### データファイル構成
```
content/
├── company-info.yaml      # 会社基本情報
├── topics.yaml           # トピックス・ニュース
├── sales-countries.yaml  # 販売国データ
├── overseas-sales.yaml   # 海外販売実績
├── manufacturers.yaml    # 取扱メーカー情報
└── offices.yaml          # 各拠点情報 (予定)
```

### 更新フロー
1. YAMLファイル編集
2. Gitプッシュ
3. 自動デプロイ (Vercel)
4. サイト更新完了

## パフォーマンス要件

### Core Web Vitals
- **LCP (Largest Contentful Paint)**: < 2.5秒
- **FID (First Input Delay)**: < 100ms
- **CLS (Cumulative Layout Shift)**: < 0.1

- **画像最適化**: Next.js Image コンポーネント
- **コード分割**: Dynamic imports
- **CSS最適化**: Tailwind CSS purging
- **キャッシュ戦略**: Vercel Edge Network
